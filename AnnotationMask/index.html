Instructions

Dieses Projekt richtet sich an Worker, die der deutschen Sprache mächtig sind. Für eine Liste von Wörtern werden verschiedene Oberbegriffe (Labels) angezeigt. Aufgabe ist es, diese Labels auf einer Skala von 0-3 zu bewerten.

Ihnen wird eine Liste von 10 Wörtern angezeigt, die zu einem Thema gehören. Zu diesem Thema werden 20 Oberbegriffe (Labels) vorgeschlagen. Sie sollen bewerten, ob die vorgeschlagenen Labels das Thema eher gut oder eher schlecht repräsentieren. Repräsentieren heißt hier: Das Label wäre eine geeignete Überschrift für dieses Thema.

Die Bewertung erfolgt mittels der folgenden Skala:

  • 3 = sehr gutes Label (man könnte mit diesem Label die meisten Wörter der Liste erraten)
  • 2 = angemessenes Label (man könnte mit diesem Label einige Wörter der Liste erraten, aber es ist zu generell oder zu spezifisch)
  • 1 = verwandt, aber kein guter Oberbegriff (der Begriff könnte in der Themenliste auftreten, stellt aber keine Überschrift dar)
  • 0 = völlig unpassendes Label (kein Bezug zum Thema)

 

Beispiel:

Thema: "Weltraum, Erde, Mond, Wissenschaft, Licht, Mission, NASA, Weltall, Rakete, Mars"

  • Raumfahrt: 3 (sehr gutes Label)
  • Weltraum: 2 (angemessenes Label)
  • Saturn: 1 (verwandt, aber kein Oberbegriff)
  • Katze: 0 (kein Bezug zum Thema)

Es ist möglich, dass es für einige Themen keine "sehr guten" Labels gibt. In diesem Fall würde kein Label die Bewertung 3 erhalten. Ebenso ist es möglich, dass für manche Themen mehrere Labels als "sehr gut" betrachtet werden können und somit die Bewertung 3 verdienen.

 

Hinweis:

Alle Labels eines Themas müssen bewertet werden, damit die Arbeit anerkannt wird. Die meisten Themenbegriffe sind aus Alltag und Medien bekannt. Falls Sie mit einem Begriff doch einmal nicht vertraut sind, ist eine kurze Websuche durchaus erlaubt.
